Sketch Wireframing Kit — v1.3

This is a collection of symbols and text styles for quickly mocking up web apps, created by your friends at Bohemian Coding, and released under the MIT license.

Document Structure

The document has 2 pages as follows:

  1. Welcome page (this one), with some default artboards for Responsive Web Design
  2. A collection of all symbols available in the document

Symbols - Usage

Use the toolbar Symbols icon to add symbols from the predefined library to your document

We have grouped them for your convenience, so you´ll have no trouble finding the perfect symbol for your design.

Text Styles - Usage

Use the toolbar Styled Text icon to add styled text layers from the predefined library to your document.

To update text layers that are using a shared text style, simply click the sync arrows that appear to the right of the style’s name in the Inspector. To reset unsynced changes, hold down the Option key and click on the reset arrow that appears in its place.
